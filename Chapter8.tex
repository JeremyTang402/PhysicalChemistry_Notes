\subsubsection{原电池和电解池}

\begin{enumerate}
    \item 电化学主要是研究电能和化学能之间的相互转化及转化过程中有关规律的科学。
    \item 电化学的用途：电解、电池、电化学分析、生物电化学。
    \item 能导电的物质称为导电体，简称导体。
    \begin{enumerate}
        \item 第一类导体：电子导体，如金属、石墨及某些金属化合物。 \\ 依靠自由电子作定向移动而导电；导电过程中导体本身不发生变化；温度升高，电阻也升高；导电总量全部由电子承担。
        \item 第二类导体：离子导体，如电解质溶液、熔融电解质等。 \\ 依靠离子的定向运动（定向迁移）而导电；导电过程中有化学反应发生；温度升高，黏度降低、离子水化作用减弱等，离子运动速率加快，电阻下降；导电总量分别由正、负离子分担。
    \end{enumerate}
    \item 原电池：化学能转化为电能的装置，自发化学反应引起电子流动。 \\ 电解池：电能转化为化学能的装置，电能驱动电极上的化学反应。生产上的应用：熔盐电解，如稀土、\ce{Ti}、\ce{Al}、有色金属。 
    \item 在原电池中，阴极是正极，阳极是负极；在电解池中，阴极是负极，阳极是正极。
    \item 电极上的反应，与电解质的种类、溶剂的性质、电极材料、外加电源的电压（电子能级，填充到离子空轨道）、离子浓度及温度等有关。类似于窗口效应：反应物快速到达表面，生成物快速离开表面。
    \item Daniell 电池：\ce{Zn(s)} | \ce{ZnSO4(aq)} || \ce{CuSO4(aq)} | \ce{Cu(s)} \\ \ce{Zn} 极产生自由电子，外电路促使电子流动，\ce{Cu^{2+}} 消耗电子。

    \item $ t_+ + t_- = 1 $，$ t_+ $ 和 $ t_- $ 分别是正离子和负离子对电流的贡献率，称为迁移数。

\subsubsection{Faraday 电解定律}

\begin{law} 
    Faraday 电解定律 (Faraday's law of electrolysis)：通电于电解质溶液之后，
    \begin{enumerate}
        \item 在电极上（两相界面上）物质发生化学变化的物质的量与通入的电荷量成正比;
        \item 若将几个电解池串联，通入一定的电荷量后，在各个电解池的电极上发生化学变化的物质的量都相等。
    \end{enumerate}
\end{law}
\begin{note}
    通入 \qty{1}{\mole} 电子，电极上各发生 \qty{1}{\mole} 化学反应，溶液中阴阳离子总共承担 \qty{1}{\mole} 电荷迁移。
\end{note}

\begin{enumerate}
    \item Faraday 常数：数值上等于 \qty{1}{\mole} 元电荷的电量， 元电荷电量 $ e = \qty{1.6022E-19}{\coulomb} $。
    
    Faraday 常数 $ F = e N_{\text{A}} = \qty{1.6022E-19}{\coulomb} \times \qty{6.022E23}{\per\mole} = \qty{96484.6}{\coulomb\per\mole} $

    \item 若在电解池中发生如下反应：
    \ce{M^{z+} + $z_{+}$ e^{-} -> M(s)}

    若反应进度为 $\xi$ ，需通入的电荷量为 $ Q = z_+ F \xi $。

    通入任意电荷量 $ Q $，则沉积出金属 B 的物质的量 $ n_{\text{B}} $ 和质量 $ m_{\text{B}} $ 分别为
    \begin{equation}
        n_{\text{B}} = \frac{Q}{z_{+} F} \qquad m_{\text{B}} = n_{\text{B}} M_{\text{B}} = \frac{Q}{z_{+} F}M_{\text{B}}
    \end{equation}

    \item 在电路中串联一个电解池，根据电解池中在阴极上析出金属的物质的量来计算通入的电荷量，这种装置就称为电量计或库仑计（coulometer）。
    
    \item 电流效率/Faraday 效率
    
    当析出一定数量的某物质时：
    \begin{equation*}
        \eta = \frac{\text{按 Faraday 电解定律计算所需理论电荷量}}{\text{实际所消耗的电荷量}} \times 100\% = \frac{n \cdot F}{I \cdot t} \times 100\%
    \end{equation*}
    
    当通过一定电荷量后：
    \begin{equation*}
        \eta = \frac{\text{电极上产物的实际质量}}{\text{按 Faraday 电解定律计算应获得产物的质量}} \times 100\%
    \end{equation*}

    \item Faraday 过程：电极上有电子交换的过程。
    
    非 Faraday 过程：电极上没有电子交换的过程，离子迁移到电极，形成电荷双层。相当于电容器，给电极充电。此时电流称为充电电流 $ I_{\text{c}} $。

\end{enumerate}

\subsubsection{离子的电迁移现象}

\begin{enumerate}
    \item 离子迁移：离子在外电场作用下发生定向运动的现象；
    
    离子扩散：离子在浓度梯度作用下发生无规则运动的现象。

    \item 当通电于电解质溶液后，溶液中正、负离子分别向两极移动，并在电极界面上发生氧化还原作用，从而两极旁溶液的浓度也发生变化。

    \item 离子本性决定离子迁移速度：离子电荷数越大、半径越小、水化作用越弱等，迁移速度越快；溶剂性质也会影响离子迁移速度：粘度等。

    \item 设想在两个惰性电极之间的溶液中，想象平面 $ AA' $ 和 $ BB' $ 将溶液分为阳极部、中部及阴极部三个部分。未通电前，各部均含有一价正、负离子各 \qty{5}{\mole} ，分别用 $ + $、 $ - $ 号代替。当通入 \qty{4}{\mole} 电子的电量时，阳极上有 \qty{4}{\mole} 负离子氧化，阴极上有 \qty{4}{\mole} 正离子还原。两电极间正、负离子要共同承担 \qty{4}{\mole} 电子电量的运输任务。  

    \begin{enumerate}
        \item 正、负离子迁移的速率相等，$ r_{+} = r_{-} $，则各分担 \qty{2}{\mole} 的导电任务。在 $ AA' $ ，$ BB' $ 平面上各有 \qty{2}{\mole} 正、负离子相互逆向通过。
        
        通电完毕后，中部溶液的浓度没有变化，而阴极部和阳极部溶液的浓度虽然相同，但与原溶液相比，正、负离子各少了 \qty{2}{\mole} 。

        \begin{figure}[h]

        通电完毕后，中部溶液的浓度仍保持不变，但阴极部和阳极部正、负离子的浓度互不相同。且两极部溶液的浓度比原溶液都有所下降。

    \item 离子电迁移的规律：
    \begin{enumerate}
        \item 向阴、阳两极迁移的正、负离子所传导的电荷量的总和恰好等于通入溶液的总电量。
        \item $\displaystyle \frac{\text{阳极部物质的量的减少}}{\text{阴极部物质的量的减少}} = \frac{\text{正离子所传导的电荷量} \ Q_+}{\text{负离子所传导的电荷量} \ Q_-} = \frac{\text{阳离子迁移速率} \ r_+}{\text{阴离子迁移速率} \ r_-} $
    \end{enumerate}
\end{enumerate}

\subsubsection{离子的电迁移率和迁移数}

\begin{enumerate}
    \item 离子在电场中运动的速率还与电场的电位梯度有关。离子的运动速率可写为：
    
    \begin{equation}
        r_{+} = u_{+} \frac{\mathrm{d}E}{\mathrm{d}l} \qquad r_{-} = u_{-} \frac{\mathrm{d}E}{\mathrm{d}l}
    \end{equation}

    式中 $ u_{+} $ 和 $ u_{-} $ 为离子电迁移率/离子淌度，当于单位电位梯度（\qty{1}{\volt\per\meter}）时离子的运动速率，单位为 \si{\meter\squared\per\volt\per\second}，大小与温度、浓度等因素有关。

    \item 正、负离子移动的速率不同，所带电荷不等，因此它们在迁移电荷时所分担的份额也不同。
\end{enumerate}

\begin{definition}
    把离子 B 所运载的电流与总电流之比称为离子 B 的迁移数（transference number），用符号 $ t_{\text{B}} $ 表示。
    \begin{equation}
        t_{\text{B}} \xlongequal{\text{def}} \frac{I_{\text{B}}}{I}
    \end{equation}
\end{definition}

\subsection{电解质溶液的电导 20260309 20260311}

\subsubsection{电导、电导率、摩尔电导率}

\begin{enumerate}
    \item 电解质导电能力用电导（electric conductance）$ G $ 表示，$ G = \frac{1}{R} $，单位为 \si{\siemens} 或 \si{\per\ohm}。
    \item 据 Ohm（欧姆）定律，电压、电流和电阻三者之间的关系为
    \begin{equation}
        R = \frac{U}{I}
    \end{equation}

    式中 $ U $ 为外加电压（单位为 V）；$ I $ 为电流强度（单位为 A）。由 $ G = R^{-1} $，可得

    \begin{equation}
        G = R^{-1} = \frac{I}{U}
    \end{equation}

    式中 $ \rho $ 是比例系数，称为电阻率 (resistivity)，单位是。 电导率 (electric  conductivity) K 是电阻率的倒数，即
    \begin{equation}
        k = \frac{1}{\rho} = \frac{L}{S R} = \frac{G L}{S} %k?
    \end{equation}

    表示：两个电极面积为\qty{1}{\meter\squared}，电极间距为\qty{1}{\meter}，包含\qty{1}{\meter\cubed} 电解质溶液时的电导。

    \item 摩尔电导率 (molar conductivity) $ \Lambda_{\text{m}} $ 把含有 \qty{1}{\mole} 电解质的溶液置于  间距为单位距离的电导池的两个平行电极之间时所具有的电导，单位为 \si{\siemens\meter\squared\per\mole}。
    \begin{equation}
        \Lambda_{\text{m}} \xlongequal{\text{def}} k V_\text{m} = \frac{k}{c}
    \end{equation}
\end{enumerate}

\subsubsection{电导率、摩尔电导率与浓度的关系}

\begin{enumerate}
    \item 强电解质溶液的电导率随浓度的增大（即导电粒子数的增多）而升高，但当浓度增大到一定程度以后，由于正、负离子之间的相互作用力增大，因而使离子的运动速率降低，电导率反而降低。在电导率与浓度的关系曲线上可能会出现最高点。
    \item 弱电解质溶液的电导率随浓度的增大而升高，但变化不显著，因为浓度增加使其解离度减小，所以溶液中离子数目变化不大。
    \item 摩尔电导率随浓度的下降而升高，因为粒子之间的相互作用力减弱，正、负离子的运动速率增大。
    \item 弱电解质摩尔电导率随浓度的下降而迅速升高，因为解离度增大，溶液中离子数目增多。
    \item 浓度极稀时（浓度在\qty{0.001}{\mole\per\liter} 以下），强电解质的 $ \Lambda_{\text{m}} $ 与 $ \sqrt{c} $ 几乎成线性关系：
    \begin{equation}
        \Lambda_{\text{m}} = \Lambda_{\text{m}}^{\infty} (1 - \beta \sqrt{c})
    \end{equation}
    \item 无限稀释：例如氯化银沉淀上清液、人体内金属有机物。
\end{enumerate}

    \item Debye-Hückel 的极限定律

    \item  Debye-Hückel 极限定律的常用表达式
\end{enumerate}

\subsubsection{Debye-Hückel-Onsager 电导理论}

\begin{enumerate}
    \item 弛豫效应（relaxation effect）
    
    \item 电泳效应（electrophoretic effect）：溶液中离子溶剂化，在外加电场作用下，溶剂化的中心离子与溶剂化的离子氛中的离子向相反方向移动，增加了粘滞力，阻碍了离子的运动，从而使离子的迁移速率和摩尔电导率下降。
    
    \item Debye-Hückel-Onsager 电导公式
    \begin{equation}
        \Lambda_\text{m} = \Lambda_\text{m}^{\infty} - (p + q\Lambda_\text{m}^{\infty}) \sqrt{c}
