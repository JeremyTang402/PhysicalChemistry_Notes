# Introduction

This repository contains my personal study notes on **Physical Chemistry**.

The notes are written in **LaTeX** and compiled using **XeLaTeX**, providing clear mathematical typesetting and well-structured scientific documentation.

These materials are continuously updated as part of ongoing study and learning.

---

# Contents

The notes include topics such as:

* Thermodynamics
* Statistical Mechanics
* Chemical Kinetics
* Quantum Chemistry
* Additional topics in physical chemistry

Many of them may have not been finished yet.

The structure of the notes will continue to expand as the notes develop.

# Compilation

These notes are compiled using **XeLaTeX**.

Compile with:

```bash
xelatex main.tex
```

or using `latexmk`:

```bash
latexmk -xelatex main.tex
```

You will need a LaTeX distribution installed, such as:

* TeX Live
* MiKTeX
* MacTeX
